visualization of the data

  1. Drawing a bar graph showing the different distributions of employees categories across regions
  1. Converted the data to numerical data
  2. Created a bar graph
subset_data <- subset_data %>%
  mutate_at(vars(all_employees, women, persons_in_a_visible_minority_group, persons_with_disabilities, aboriginal_peoples),
            as.numeric)

subset_data_long <- subset_data %>%
  gather(key = "category", value = "value", -type_of_employment)

ggplot(subset_data_long, aes(x = type_of_employment, y = value, fill = category)) +
  geom_bar(stat = "identity", position = position_dodge(width = 0.8)) +
  labs(title = "Type of Employment by Population distribution",
       y = "Population") +
  theme_minimal() +
  theme(axis.text.x = element_text(angle = 45, hjust = 1)) +
  scale_y_continuous(limits = c(0, 16000), breaks = seq(0, 16000, by = 1000))

from the bar graph above you can see:

  • there are more indeterminate women employees compared to other employment types
